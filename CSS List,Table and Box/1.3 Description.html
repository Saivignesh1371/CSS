HTML
Hypertext Markup Language
HTML element
HTML element is a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document. HTML elements can contain formatting instructions, semantic meaning, and content.
HTML tag
0HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content.
Angle Bracket
The "<" and ">" symbols that are on the comma and period keys on the keyboard. Angle brackets are commonly used to enclose a code of some type.
HTML attribute
An HTML attribute is a piece of markup language used to adjust the behavior or display of an HTML element. For example, attributes can be used to change the color, size, or functionality of HTML elements.
HTML Comment
The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
HTML Object Element
The External Object element. TheHTML element represents an external resource, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.
HTML Special character
A special character is a character that is not an alphabetic or numeric character. Punctuation marks and other symbols are examples of special characters.
